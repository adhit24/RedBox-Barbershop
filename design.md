# Design System — Redbox Barbershop Website
## Inspired by Reydy Barbershop UX System

---

## 1. Design Philosophy

Website bukan sekadar informasi, tapi:
> **Tool untuk booking + experience preview**

Fokus utama:
- Visual kuat
- Flow cepat
- Conversion tinggi

---

## 2. Core UX Flow

Struktur utama (mengikuti sistem Reydy):

1. Landing (impact)
2. Services (pilih layanan)
3. Barber (pilih orang)
4. Time Slot (pilih waktu)
5. Confirm booking

📌 Insight:
User diarahkan step-by-step tanpa mikir  
→ ini bikin conversion tinggi

---

## 3. Layout Structure

### Homepage Structure

**1. Hero Section**
- Full width image/video
- Headline besar

Contoh:
“Sharp Cuts. Bold Style.”

- CTA:
  - Book Now
  - View Services

---

**2. Services Preview**
- Grid / tab system
- Category:
  - Haircut
  - Beard
  - Package

---

**3. Barber Section**
- List barber
- Card style:
  - Foto
  - Nama
  - CTA: Book

---

**4. Booking Section**
- Step-by-step explanation:
  1. Select service
  2. Pick barber
  3. Choose time
  4. Confirm

---

**5. Gallery**
- Slider / grid
- Fokus hasil haircut

---

**6. Location & Contact**
- Maps
- Jam buka
- WhatsApp CTA

---

## 4. UI Components

### Button System

Primary:
- Background merah
- Text putih
- Rounded medium

Secondary:
- Outline putih / abu
- Transparent

---

### Card System

**Barber Card:**
- Image
- Name
- Service count
- CTA button

**Service Card:**
- Title
- Duration
- Price
- CTA

---

### Booking UI

- Calendar picker
- Time slot grid
- Disabled slot (booked)
- Active slot (available)

---

## 5. Visual Style

### Color

- Primary: Red
- Background: Black / Dark Grey
- Text: White
- Accent: Grey / subtle gold

---

## Brand Assets
- Always check the `brand_assets/` folder before designing. It may contain logos, color guides, style guides, or images.
- If assets exist there, use them. Do not use placeholders where real assets are available.
- If a logo is present, use it. If a color palette is defined, use those exact values — do not invent brand colors.

---

### Typography

- Bold headline
- Sans-serif modern
- Clean hierarchy

---

### Imagery

- Close-up haircut
- Barber in action
- Before/after

---

### Lighting Style

- High contrast
- Dramatic shadow
- Focus highlight

---

## 6. Interaction Design

- Smooth scroll
- Hover effect:
  - Button → glow / lighten
  - Card → slight zoom

- Transition:
  - Fast (0.2–0.3s)
  - Clean

---

## 7. Booking Experience (CRITICAL)

Mengikuti sistem Reydy:

### Step Flow:

1. Select Service  
2. Select Barber  
3. Select Time  
4. Confirm  

📌 Harus:
- Tidak lebih dari 3 klik per step
- Tidak membingungkan

---

### Slot System UI

- Grid jam:
  - 10:00
  - 11:00
  - 12:00

Status:
- Available → normal
- Booked → disabled
- Selected → highlight merah

---

## 8. Mobile Experience

Prioritas utama (mobile-first)

- Sticky CTA “Book Now”
- Bottom navigation (optional)
- Scroll pendek
- Thumb-friendly

---

## 9. Conversion Elements

- CTA di setiap section
- Booking selalu terlihat
- Social proof (gallery)

---

## 10. Differences (Redbox Adaptation)

Agar tidak copy 100%:

**Reydy Style:**
- Lebih “international premium”
- Banyak barber choice

**Redbox Adaptation:**
- Lebih lokal & relatable
- Lebih fokus experience
- Lebih “warm” tone

---

## 11. Key Principle

Website harus terasa:

- Cepat
- Jelas
- Premium
- Tidak ribet
